Project Overview

Purpose: Educational and demonstration use only.

This project is designed to help users understand: client-server architecture, secure communication, and remote command handling in a controlled environment.

Important Notice

Warning:
It is not recommended to run this software on your personal or primary system.
Always use a virtual machine or controlled test environment.

How It Works

The system is built using socket-based communication between a client and a server.
Communication is secured with SSL encryption.

The client continuously listens for commands sent by the server and executes them in real time.

Features

✔ Execute shell commands
✔ Navigate directories (cd operations)
✔ Retrieve system information
✔ Secure socket communication (SSL)
✔ Real-time command handling

Technologies Used

• Python
• Socket Programming
• SSL Encryption
